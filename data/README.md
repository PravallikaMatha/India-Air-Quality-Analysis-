# Data

The original air quality dataset (`Data.csv`) is not included in this repository because it exceeds GitHub's file size limit.

The analysis was performed using publicly available datasets from the following official sources:

## Air Quality Data

Central Pollution Control Board (CPCB)

https://app.cpcbccr.com/ccr/#/caaqm-dashboard-all/caaqm-landing

## NCAP Funding

Ministry of Environment, Forest and Climate Change (MoEFCC)

https://moef.gov.in/en/division/national-clean-air-programme/

## State Demographic Data

Office of the Registrar General & Census Commissioner, India

https://censusindia.gov.in/

The processed datasets (`NCAP_Funding.csv` and `State_data.csv`) used in this repository are included for reproducibility.
